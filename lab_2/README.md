# Лабораторна робота №2 — Наука про дані: підготовчий етап

## Опис завдання
Завантаження VHI даних для областей України, data cleaning,
формування вибірок. Аналіз датасету Individual Household Electric Power Consumption.

## Вимоги
- Python 3.10+
- pip

## Встановлення залежностей
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск
Відкрити `lab2_part1.ipynb` або `lab2_part2.ipynb` у Jupyter Notebook:
```bash
jupyter notebook
```
