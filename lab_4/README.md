# Лабораторна робота №4 — Візуалізація даних 2

## Опис
Інтерактивна програма для візуалізації гармоніки з шумом та фільтрацією.

## Запуск
```bash
pip install -r requirements.txt
py lab4.py
```

## Інструкція
- Слайдери — змінюй амплітуду, частоту, фазу, параметри шуму
- Чекбокс — вмикає/вимикає відображення шуму
- Reset — повертає початкові параметри

## Вимоги
- Python 3.8+
